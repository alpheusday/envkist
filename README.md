# Envkist

A utility to defer the execution of environment variables.

## Installation

Install this package as a dependency in the project:

```sh
# npm
npm i envkist

# Yarn
yarn add envkist

# pnpm
pnpm add envkist

# Deno
deno add npm:envkist

# Bun
bun add envkist
```
